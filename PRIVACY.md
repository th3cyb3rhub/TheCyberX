# Privacy Policy - TheCyberX Browser Extension

**Last Updated:** January 2026

## Overview

TheCyberX is an open-source browser extension for security professionals and penetration testers. We are committed to protecting your privacy.

## Data Collection

**TheCyberX does NOT:**
- Collect any personal information
- Track your browsing history
- Send any data to external servers
- Store any data outside your browser
- Use analytics or tracking services

## Data Storage

All data processed by TheCyberX (such as encoded strings, hashes, or payloads) is:
- Processed entirely locally in your browser
- Never transmitted to any external server
- Not stored persistently (cleared when you close the extension)

## Permissions

TheCyberX requests the following browser permissions:

| Permission | Purpose |
|------------|---------|
| `activeTab` | Access current page for recon tools (link extraction, etc.) |
| `storage` | Save user preferences locally |
| `tabs` | Access current tab URL for analysis |
| `cookies` | Cookie editor functionality |
| `scripting` | Execute content scripts for page analysis |
| `<all_urls>` | CORS checking and header analysis on any site |

## Third-Party Services

TheCyberX does NOT use any third-party services, APIs, or analytics.
